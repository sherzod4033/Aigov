# Scripts

## 10-minute stability check

Run:

```bash
./scripts/stability_check.sh
```

Optional environment variables:

- `BASE_URL` (default `http://localhost:8001`)
- `API_PREFIX` (default `/api/v1`)
- `STABILITY_USERNAME` (default `admin`)
- `STABILITY_PASSWORD` (default `admin123`)
- `DURATION_SECONDS` (default `600`)
- `INTERVAL_SECONDS` (default `5`)
